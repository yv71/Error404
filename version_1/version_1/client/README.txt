Vous devez joindre avec votre client un fichier readme expliquant comment compiler (si besoin) et lancer votre client.
Ce fichier README présente un exemple (adaoté au client fourni) sur lequel vous basez pour votre fichier README.

***
Groupe : Enseignant
Langage utilisé : JAVA

***
- compilation du client
(sur cet exemple, les sources ne sont pas fournies - mais important à préciser pour votre client)

javac Main.java 

***
- lancement par défaut (en localhost et port 1337):

java -jar clientfruit.jar

***
- lancement avec options (ip puis port)

java -jar clientfruit.jar 127.0.0.1 1337

***
- acces aux options (si possible)

java -jar clientfruit --help

***
